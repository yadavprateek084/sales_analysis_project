# Sales Dashboard – Power BI Project

## 📌 Project Overview
This project is an interactive **Sales Dashboard** built using **Power BI** to analyze sales performance across multiple years, regions, customers, and product categories. The dashboard provides actionable business insights through dynamic visualizations and KPI tracking.

The dashboard helps in:
- Monitoring revenue and sales quantity
- Analyzing market-wise performance
- Tracking yearly and monthly sales trends
- Identifying top customers
- Understanding product category contribution

---

## 🚀 Features
- Interactive year-wise filtering (2017–2020)
- Regional filtering (Central, North, South)
- Revenue and sales quantity KPIs
- Market-wise revenue analysis
- Market-wise sales quantity analysis
- Monthly revenue trend visualization
- Top 5 customers revenue analysis
- Product type sales quantity comparison
- Fully interactive slicers and charts

---

## 🛠️ Tools & Technologies Used
- Power BI

---

## 📈 Key Insights
- Delhi and Mumbai generated the highest revenue in overall sales.
- Revenue showed fluctuations across different years.
- Own Brand products contributed the highest sales quantity.
- Top customers generated a significant portion of total revenue.
- Regional filtering helps identify high-performing markets.
